# HR_Analytics_Dashboard

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** developed using Power BI to analyze employee data and generate insights on workforce distribution, attrition, performance, engagement, training, and diversity to support data-driven HR decisions.

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV Dataset

---

## 📂 How to Load Dataset (CSV)

1. Open Power BI Desktop
2. Click on **Get Data → Text/CSV**
3. Select the dataset file
4. Click **Load** (or Transform Data if preprocessing is needed)
5. Create relationships (if multiple tables)
6. Start building visuals and dashboards

---

## 📊 Dataset Information

The dataset contains employee-related information such as:

* Employee ID, Age, Gender, Department, Job Role
* Salary, Bonus, Experience, Tenure
* Performance Rating, Job Satisfaction, Work-Life Balance
* Training Hours, Promotions, Attrition
* Engagement Score, Overtime, Business Travel

---

## ⭐ Key Features

* Workforce Overview (Employees, Salary, Experience)
* Attrition Analysis (Department, Overtime, Salary impact)
* Performance & Salary Insights
* Employee Engagement Analysis
* Training & Career Growth Analysis
* Diversity & Inclusion Insights
* Interactive filters and slicers for dynamic analysis

---

## 📈 Insights

* Workforce is balanced with good diversity
* Attrition is higher among employees with overtime and low work-life balance
* Training programs positively impact performance and engagement
* Lack of promotions increases employee turnover
* Salary and performance show moderate alignment
